# Supply Chain Performance Analytics Dashboard

## Project Overview
Built an interactive Power BI dashboard analyzing supply chain operations, including sales performance, profit analysis, shipping efficiency, and customer segmentation. The dashboard provides actionable insights to support data-driven decision-making and operational optimization.

## Key Features
- Sales and revenue trend analysis
- Profitability analysis by product, region, and category
- Shipping performance and delivery tracking
- Customer segmentation and performance insights
- Interactive filters and dynamic visualizations

## Technologies Used
- **Power BI**
- **DAX (Data Analysis Expressions)**
- **Power Query (Data Cleaning and Transformation)**

## Business Impact
- Enabled real-time monitoring of supply chain KPIs
- Identified high-performing and underperforming segments
- Improved visibility into operational efficiency
- Supported strategic planning through data-driven insights
